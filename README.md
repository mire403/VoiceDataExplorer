# VoiceDataExplorer
VoiceDataExplorer（语音数据智能挖掘）是一个以语音为第一输入模态的数据探索系统。它把大量非结构化语音转成可查询、可聚合、可追溯到秒级的决策级数据资产。系统先对音频做语音识别，输出带说话人和时间戳的标准化话语片段；再对片段做语义事件抽取，识别决策点、KPI指标、责任人、时间约束、行动项和风险担忧；然后对实体）和关系做显式建模，构建事件图和时间线。用户可以用自然语言做三类查询：上周会议中与KPI相关的所有决策、某个KPI被讨论的次数与趋势、尚未形成决策的风险点。每一步输出都是结构化JSON，每个事件都能追溯到原始语音的时间戳；LLM只用于分类与抽取，不做开放式总结。MVP 面向1–2小时会议、约50–200条话语，支持时间线导出为Markdown与HTML，便于阅读与二次分析。
